Everdeck Library

A web app for archiving and searching through the games playable by the Everdeck. The app was built by a three person team in Next.js with a MongoDB backend and deployed on Vercel. Using the app, you can use searches or filters to find a game you want to play and find the rules and card mappings for that game. The app can be found at everdeck-library.vercel.app. If you want more information about the Everdeck, check out the about page of our app here.

Features

A card system to display basic information for the game on the main page of the app.

The card contains the game's title, a short description, stats (player count, play time, and complexity), and a warning icon if the game has any warnings. The card also contains the game's box art if the user is on a bigger screen.

Search and Filter

If you know the name of the game you want to play, you can use the search bar at the top of the main page. You can also sort the games currently on the main page in ascending or descending order by using the buttons located below the search bar.

If you know some of the qualities of a game you want to play but don't know the exact game (ex. a game for 4 players), you can use the filter menu.

The filter menu has sliders to choose the minimum and maximum values for player count, play time, and complexity.

There is a dropdown to choose a mapping strength of a game which represents how well the game maps to the Everdeck. There are 4 options: "Perfect", "High", "Low", and "Any". (More information on mapping strength can be found on the "How To" on the Everdeck Library app which can be accessed by clicking the "?" button on the bottom left of the page)

Checkboxes are provided to choose external components (things in addition to the Everdeck that a game requires to play. ex. Dice) and game types.

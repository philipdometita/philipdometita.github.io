Pak Patch Manager

A simple Python program for managing modification files for the game Monster Hunter: Rise which makes adding and removing mods a little less painful and little more convient.

First, a background. There are multiple ways to mod the game Monster Hunter: Rise (referred to as Rise from here forward). One of those ways is by adding a pak patch file to the games directory. These files have the name re_chunk_000.pak.patch_XYZ.pak, where XYZ starts at 001 and is incremented for every pak patch file added.

Pak file structure

Given enough mods and enough time, it would be difficult to know which mod is which when you want to remove or update a mod. The simple solution is to keep a note about which pak file relates to which mod and update the note every time you add or remove mods.

Pak Patch Manager does one better by keeping track of mods while placing or removing the mods in Rise's directory while renaming the files so the file names remain consistent.

Features

Add and remove specific mods from Rise's main directory

PPM activating mod #3

remove all currently activated mods from Rise's main directory. Especially useful if the game is updated and mods are not compatible with the new version.

PPM deactivating all active mods

Read the mods folder again so you can add mods while using the program without having to restart it.

PPM re-reading mods folder

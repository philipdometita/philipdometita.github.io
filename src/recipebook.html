Recipe Book

An android app built in React Native with the purpose of consolidating recipes that you want to try or have enjoyed. Recipe Book was inspired by an old and beat-up notebook that my mother recorded many recipes for the foods of my childhood. I made Recipe Book with the intention of helping me get better at cooking and React Native at the same time.

Features

Ability to save the details of a recipe such as, ingredients, directions, cook time, and servings made. You can edit the details of the recipe after it is saved.

Add / edit recipe screen

Delete saved recipes by touching and holding on the recipe you want to delete.

Delete pop up

Delete confirmation menu

A tag system to better categorize recipes and make recipes of a specific type easier to find.

Add tags menu

Displayed tags on recipe screen

Filter added recipes by tags and Sort recipes by cook time, servings, or date added to the app.

Filter menu

Search for recipes using keywords that are in the title or description.

Search by keyword in title

Search by keyword in description
